API调用示例 

curl -X POST "https://api.whisperai.com/v1/recognize" \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: multipart/form-data" \
-F "audio=@/path/to/audio.mp3" \
-F "task=transcribe" \
-F "language=zh"
